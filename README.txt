== Description ==
With Realty Workstation real estate brokers and team leaders can to process their agents’ transactions, archive all transaction related documentation and calculate commissions. Create a profile for each agent on your team, with a unique payout for each agent. Get alerts when a new transaction is complete and submitted for payment by the agent. Automatically calculate commission payments. Have all contracts and documentation easily stored for each transaction at the same time that it is uploaded and submitted by the agent.

== Frequently Asked Questions ==
= Do my agents need to access my WordPress dashboard to use Realty Workstation? =
No, agents can access Realty Workstation through a unique URL that provides web access to the workstation.

= How many agents can I manage with Realty Workstation? =
The free version of Realty Workstation allows brokers and team leaders to create 5 agent profiles. For larger teams a premium version of Realty Workstation is available on [realtyworkstation.com](realtyworkstation.com)

= Where can I find tutorials videos on how to install and manage Realty Workstation? =
Tutorials are available on [realtyworkstation.com/tutorials](realtyworkstation.com/tutorials)
